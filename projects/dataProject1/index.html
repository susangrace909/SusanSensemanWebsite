Retail & Sales KPI Dashboard

A Power BI Project exploring sales performace, profitability, and regional insights using the Superstore dataset.

  • Role: Data Analyst
  • Tools: Power BI, Excel
  • Date: 2026

Watch the Dashboard Demo

A short walkthrough of the Power BI dashboard, showcasing KPI tracking, regional insights, and interactivity.

Problem

Leadership teams needed a clear, centralized view of sales performance that connected KPIs, category trends, and regional results.

Approach

Using the Superstore dataset from Kaggle, I modeled and cleaned the data in Power BI, defining relationships and DAX measures for key metrics like year-over-year growth and profit margin. I designed an interactive dashboard featuring KPI cards, category and product visuals, and a regional map to make insights easy to explore from multiple perspectives.

Outcome

The final dashboard turns raw sales data into a cohesive, actionable story. Users can drill down by product, category, or region, and reset filters for a consistent viewing experience. The project demonstrates how Power BI can streamline decision-making by replacing manual reports with dynamic, data-driven visuals.

Highlights

  • Designed KPI cards and trend charts with consistent visual hierarchy
  • Built bookmark-based navigation for intuitive filter resets
  • Applied DAX for profitability, margin, and time-series analysis
  • Created branded, cohesive layout aligned with my personal design style
