Sales Dashboard

Turning monthly sales data into an interactive, decision-ready dashboard.

  • Role: Data Analyst
  • Tools: Power BI, SQL, Python (pandas), Excel
  • Date: 2024
  • Links: Live Demo · GitHub

Problem

Leadership needed a single source of truth for revenue, margins, and regional performance with drilldowns by product and customer segment.

Approach

Cleaned and modeled data (SQL + Python), designed a star schema for BI, and built a Power BI dashboard with DAX measures (YoY, MoM, cohort retention).

Outcome

Reduced manual reporting time by 70% and surfaced actionable insights (e.g., underperforming SKUs by region), guiding quarterly pricing and inventory decisions.

Highlights

  • KPI cards and drill-through pages
  • Bookmark-based “what-if” scenarios
  • Row-level security for regional managers
